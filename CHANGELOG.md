All notable changes to this project will be documented in this file.

and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.1] - 2024-12-30

### Changed
- Updated README with NPM package installation instructions
- Added build command documentation

## [0.2.0] - 2024-12-30

### Changed
- Configured Vite library build for ES and UMD bundles
- Optimized package size by adding `.npmignore`
- Added TypeScript declaration files and source maps

### Added
